Speedhack can be used with ExigoHelper.exe

# Installation:
1. Install CheatEngine.

Import content of the file "Speedhack.CT" to CheatEngine and export it as .exe file.

2. Edit paths in ExigoHelper\eh_devtools\DevTools.conf as according to new speedhack file executable. The exe name will be read from config file via this line: DevToolsExigoSpeedhackInitiatorExecutableName.

3. Place new .exe file into game folder and run it.

# Note
Enjoy speedhack >> ONLY for testing purposes!

DevTools needs to be set to true!
