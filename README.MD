# Basics

## Description

Helper was created to make some common operations easier (especially switching between game versions)
by simply selecting action numbers in console .

Basic helper functions include actions:
- launch game
- launch editor
- change data version of the game (it includes available recent data files game versions)
- launch dgVoodoo configurator (you need to install it yourself!)
- enable/disable dgVoodoo

### Included data files game versions (all game versions include HD support patch):
- 1.4.02_08_test1
- 1.4.02_05_prepatch5

## Installation
- copy content of folder "app" into game folder
- open and edit file ExigoHelper.conf as according to your system paths

## Usage
- simply run ExigoHelper.exe

# Exigo Helper DevTools

## Description

DevTools provides tools used to build data ORK files.

This includes:
- build DataX.ork
- run game with X SpeedHack
- build DataX.ork AND Run game with X SpeedHack
- change SpeedHack value

## Installation
- copy folder "eh_devtools" into game folder
- adjust DevTools.conf paths and enable it by changing DevTools line to "true" value.

## Usage
New options will be available in ExigoHelper.exe
